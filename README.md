# แผนชีวิตของคุณ | คุยไป คลิกไป

## โหมดใช้งานจริง (แนะนำ)
**นักวิเคราะห์แผน AI · ลูกค้าคลิกไปด้วย · เห็นหน้าเราบนจอ (วิดีโอวงกลม)**  
ไม่โชว์บทพูด · เสียงหุ่นยนต์ (TTS) **ปิดเป็นค่าเริ่มต้น**

1. วางไฟล์ `avatar.mp4` (วิดีโอวนลูปวงกลม) คู่กับ `index.html`
2. เปิด **index.html** ด้วย Chrome / Edge / Safari
3. แชร์จอ / วางแท็บเล็ตให้ลูกค้าเห็น
4. คลิปอธิบาย · ลูกค้าเลื่อน/กดเลือก

ปุ่ม 🔇/🔊 = เปิด–ปิดเสียงวิดีโอ

## วิธีใช้ด่วน
1. เปิดโฟลเดอร์ `life-timeline` → เปิด **index.html**
2. ใช้หน้าจอมือถือ/แท็บเล็ตแนวตั้งดีที่สุด
3. ไฟล์วิดีโอ: `avatar.mp4` (poster สำรอง `avatar.png`)

## คุณสมบัติ
- วิดีโอ avatar วนลูป ครอปวงกลม ตลอด flow
- โหมดมีต: คุยไป คลิกไป · ไม่บังคับอัดเสียง/ท่องบท
- TTS เป็นทางเลือก (ปิดเริ่มต้น)
- ไม่ต้องพิมพ์ · เลื่อน slider / กด chip
- สรุป personalised + priority
- Dark premium UI (ทอง + cyan)
- Confetti ตอนจบ · ข้อมูลอยู่แค่ในเครื่อง

## ขั้นตอน (10 steps)
0. Landing + Hook
1. อายุ
2. เป้าหมายเกษียณ (อายุ + ใช้เดือนละ)
3. คำนวณเงินที่ต้องเตรียม + เตรียมไว้แล้ว
4. ครอบครัว (ดูแลใคร + บุตร)
5. เป้าหมายชีวิต (multi)
6. ความเสี่ยงที่กังวล
7. พีระมิดการเงิน + เงินสำรอง
8. ผล EF + ประกันที่มี
9. ความเสี่ยงลงทุน + มรดก
10. สรุป + Priority + CTA นัดปรึกษา
