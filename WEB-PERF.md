# เกณฑ์เว็บจริง — เบา · โหลดเร็ว · กดเล่นได้

เป้าหมาย: deploy เป็นหน้าเว็บ (static) ให้ลูกค้าเปิดจากมือถือ/แชร์ลิงก์ได้  
**ไม่ใช่ไฟล์โปรเจกต์หนัก** — โฟกัสความเร็ว

---

## งบขนาดไฟล์ (ยึดตามนี้ตอนอัด/แปลง)

| รายการ | เป้า | สูงสุด |
|--------|------|--------|
| คลิปต่อหน้า | **0.8–1.5 MB** | 2.5 MB |
| หน้าแรก `avatar.mp4` | **1–2 MB** | 3 MB |
| รวมวิดีโอทั้งแอป (ถ้าครบ ~11 คลิป) | **8–15 MB** | ≤ 20 MB |
| รูป poster `avatar.png` | ≤ 80 KB | 150 KB |
| `index.html` + CSS/JS | ~50–80 KB | — |

**เปิดหน้าแรก (first paint):** เป้าโหลด HTML + poster ก่อน · วิดีโอโหลดตาม · อย่าบล็อกจอขาว

---

## สเปกวิดีโอตอนแปลง (บังคับ)

- รูปแบบ: **MP4 · H.264 + AAC**
- ความละเอียด: **สูงสุด 720px ด้านยาว** (แนวตั้ง ~720×1280 หรือครอปใกล้หน้า ~480–720)
- เฟรมเรต: 24–30 fps
- เสียง: mono AAC **96–128 kbps**
- ตัดเสียงห้องเงียบ / normalize เบาๆ ได้
- `faststart` (moov ต้นไฟล์) = เล่นบนเว็บเร็ว
- ความยาวต่อคลิป: **15–35 วินาที** (หน้าแรกไม่เกิน ~40 วิ)

คำสั่งตัวอย่าง:

```bash
ffmpeg -i input.mov -c:v libx264 -pix_fmt yuv420p -vf "scale='min(720,iw)':-2" -crf 26 -preset medium -c:a aac -b:a 96k -ac 1 -movflags +faststart out.mp4
```

- `crf 26–28` = เล็กลง (ถ้าหน้าชัดพอ)  
- อย่าใช้ HEVC/ProRes/.mov ดิบบนเว็บ

## โครงสร้างไฟล์ที่ deploy

```
life-timeline/
  index.html          # แอปหลัก
  avatar.png          # poster เล็ก
  media/
    00-landing.mp4    # หน้าแรก (หรือ avatar.mp4)
    01-age.mp4
    02-retire.mp4
    ...
```

- โหลด**ทีละหน้า** (lazy): เข้าหน้าไหนค่อยโหลดคลิปหน้านั้น  
- ไม่ preload ทั้ง 11 คลิปพร้อมกัน  
- ขั้นอื่นที่ยังไม่มีคลิป = ใช้คลิปหน้าแรกวนเงียบ หรือไม่มีวิดีโอก็ได้ชั่วคราว

---

## UX โหลดเร็ว

1. โชว์ UI + ข้อความก่อน · วิดีโอตามมา  
2. มี `poster` ทุควิดีโอ  
3. ปุ่ม “แตะเพื่อเปิดเสียง” ถ้า autoplay ถูกบล็อก  
4. คลิปเล่นจบ → หยุดหรือวน**เงียบ** (ไม่ย้ำประโยคเดิมดังซ้ำ)  
5. Host: Cloudflare Pages / Netlify / GitHub Pages / S3+CDN ก็ได้ (static พอ)

---

## สิ่งที่ไม่ทำ (เพื่อความเบา)

- ❌ คลิป 1–2 นาทีต่อหน้า  
- ❌ 4K / 1080p full bitrate  
- ❌ ไฟล์ .mov ดิบขึ้นโฮสต์  
- ❌ วิดีโอทุกหน้า autoplay พร้อมเสียงพร้อมกัน  
- ❌ ใส่ library หนัก (React bundle ใหญ่) — ตอนนี้ single HTML ดีอยู่แล้ว

---

## เช็คก่อนปล่อย

- [ ] เปิดจากมือถือ 4G หน้าแรกขึ้นใน ~2–3 วิ  
- [ ] คลิปหน้าแรก < 3 MB  
- [ ] รวม media ทั้งก้อน < 20 MB  
- [ ] เล่นได้บน Chrome Android + Safari iOS  
- [ ] ไม่มี TTS บังคับ (ใช้เสียงคลิปจริง)
